# KBO 야구 알리미 — 미니앱 개발 진행 기록

## 프로젝트 정보
- 앱 이름: KBO 야구 알리미
- 앱 유형: 비게임 (TDS 디자인 시스템 필수)
- 브랜드 컬러: #3182F6 (토스 블루, 추후 KBO 이미지에 맞게 조정 가능)
- 생성일: 2026-04-10
- 아키텍처: 하이브리드 — 본 miniapp(Vite/React)은 UI 전용, 백엔드는 상위 kbo_game(Next.js API Routes) 호출

## 핵심 컨텍스트
- 상위 kbo_game은 Next.js 16 기반의 KBO 경기 종료 알림 서비스 (Supabase + Toss Push API mTLS)
- 앱인토스는 정적 번들만 업로드 가능하므로 Next.js를 그대로 쓸 수 없음 → miniapp을 Vite로 별도 scaffold
- miniapp은 kbo_game의 `/api/*` 엔드포인트를 REST로 호출하는 구조
- 토스 OAuth 토큰은 kbo_game 서버에서만 취급, 클라이언트에는 앱 자체 JWT 발급
