## 12. 약관 등록 방법 안내

약관을 온라인에 게시하고 앱인토스 콘솔에 URL을 등록해야 합니다. 3가지 방법 중 선택하세요:

### 방법 1: 자체 웹페이지 (권장)
1. `/terms`, `/privacy` 페이지를 서비스 도메인에 생성 (정적 페이지 권장)
2. 외부 비로그인 접근 가능 확인
3. 앱인토스 콘솔의 약관 등록에서 전문 붙여넣기 또는 URL 등록
4. 필수/선택 설정 확인

### 방법 2: 노션/구글독스 "웹 게시"
1. 문서 작성 후 "웹에 게시/공개 링크" 활성화
2. 비로그인으로 열리는지 확인
3. 콘솔에 URL로 등록 또는 전문 붙여넣기

### 방법 3: GitHub 이슈로 공개 URL 만들기 (간편 MVP용)
1. **Public** GitHub 레포 선택
2. `Issues` 탭 → `New issue` 클릭
3. 제목: `서비스 이용약관` 또는 `개인정보처리방침`
4. 본문에 약관 전문을 붙여넣기 (마크다운 가능)
5. `Submit new issue` → 생성된 이슈 URL을 콘솔 약관 URL로 사용

> ⚠️ GitHub 이슈 방법은 레포가 **Public**이어야 외부에서 접근 가능합니다. Private으로 전환하면 링크가 차단되므로 주의하세요.

## 13. 연결 끊기 콜백 설정 (선택)

토스 로그인에서 추가 동의 항목을 켠 경우 또는 서버에서 계정 연동을 관리하는 경우 권장합니다.

### 권장 설정값
- **콜백 URL**: `https://{도메인}/api/toss/unlink`
- **메서드**: POST
- **인증**: Basic Auth (랜덤 user:pass 생성 → 서버 환경변수로 저장)
- **검증**: 콘솔 "테스트하기"로 200 OK 확인

### 구현 포인트
1. Basic Auth 헤더 검증
2. 요청 body에서 사용자 식별자 추출
3. 해당 사용자의 연동 정보 삭제 / 탈퇴 처리
4. 200 응답 반환

### 콘솔 등록 referrer (3종)
- `UNLINK` — 사용자가 연동 해제 시
- `WITHDRAWAL_TERMS` — 약관 철회 시
- `WITHDRAWAL_TOSS` — 토스 탈퇴 시
